
## Software Contributing Guidelines
1. Use `main` instead of `master`.
2. Naming conventions are as follows:
    - Files: `snake_case.cc`
    - Variables: `snake_case`
    - Constant variables: `kPascalCase`
    - Functions: `PascalCase`
    - Types: `PascalCase`
    - Private variables within classes: `snake_case_`
